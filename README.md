# Prova Final - Mundo 01

Script que simula um cadastro de aluno com as seguintes funcionalidades:

1. **Verificação de idade** - bloqueia menores de 18 anos
2. **Análise de nome** - exibe o nome em maiúsculo/minúsculo, conta letras, identifica primeiro e último nome, e verifica se contém "Silva"
3. **Média de notas** - calcula a média de duas notas e exibe o resultado (aprovado, recuperação ou reprovado)
4. **Pagamento de mensalidade** - simula opções de pagamento com desconto à vista, parcelamento sem juros e com juros
5. **Sorteio** - alunos com média >= 8 participam de um sorteio aleatório

## Como rodar

```bash
python prova_final.py
```

## Requisitos

- Python 3.x
